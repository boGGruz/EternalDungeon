# EternalDungeon

Простой 2D dungeon crawler / action RPG, написанный на Python с использованием Pygame. Включает процедурную генерацию уровней, базовый ИИ врагов, систему боя и предметов.


## Технологии

*   Python 3
*   Pygame

## Запуск

1.  **Клонировать:**
    ```
    git clone https://github.com/boGGruz/EternalDungeon.git
    cd EternalDungeon
    ```
3.  **Установить Pygame:**
    ```
    pip install pygame
    ```
4.  **Запустить:**
    ```
    python main.py 
    ```

## Управление

*   **Движение:** `W`, `A`, `S`, `D`
*   **Бег:** `Left Shift`
*   **Атака/Стрельба:** ЛКМ
*   **Лечение:** `X`
*   **Смена оружия:** `1` / `2`
*   **Подобрать предмет:** `F`
*   **Пауза/Меню:** `Escape`
*   **Полный экран:** `F11`
